## 发布订阅模式

```js
Class Notify {
    constructor() {
        this.subscribers = []
    }
    add(handler) {
        this.subscribers.push(handler)
    }
    emit() {
        this.subscribers.forEach(subscriber => subscriber())
    }
}

let notify = new Notify()
// 收集依赖
notify.add(() => {
    console.log('emit here');
})

// 执行任务
notify.emit()
```
