## vim命令

- 命令模式：`:命令`
- 编辑模式：`i a o进入`

## 命令模式
- `:set nu`：显示行号
- `s/old/new/g`：全部替换
- `/关键字`：：搜索 n下一个
- `dd`：剪切
- `p`：粘贴
- `ctrl+r`： 恢复
- `ctrl+w s`：  上下分屏
- `ctrl+w v`： 左右分屏
- `gg`： 光标移动到文件的第一行
- `G`：光标移动到文件的最后一行
- `:数字`： 光标移动到指定的行
- `:q`： 退出，在文件内容没有被改变时使用
- `:q!`： 强制退出不保存，在改变文件后不想保存时使用
- `:wq`： 退出保存
- `:wq!`： 强制退出保存，在对只读文件编写时使用，注意，所编辑的只读文件必须是当前用户的文件；或者当前操作的用户是root
